# Greed Snake - 运行指南

## 快速开始

### 方式一：本地服务器（推荐）

```bash
cd /home/brian/work_project/Greed_Snake
python3 -m http.server 8088
```

然后在浏览器打开 `http://localhost:8088`

### 方式二：直接打开文件

双击 `index.html` 或右键 "在浏览器中打开"。注意：部分浏览器可能限制 `file://` 协议下的 Web Audio，建议使用方式一。

### 方式三：VS Code Live Server 插件

在 VS Code 中右键 `index.html` → "Open with Live Server"

---

## 操作说明

| 按键 | 功能 |
|------|------|
| `↑` `↓` `←` `→` 或 `W` `A` `S` `D` | 控制蛇的移动方向 |
| `Space` | 开始游戏 / 游戏结束后重新开始 |
| `P` 或 `Esc` | 暂停 / 恢复游戏 |

---

## 游戏玩法

1. 按 `Space` 开始游戏，蛇向右移动
2. 吃掉粉红色的食物得分（每次 +10）
3. 躲避灰色障碍物（碰撞会让蛇变短 1 节）
4. 收集金色道具加速（1.5×），蓝色道具减速（0.75×），持续 5 秒
5. 每通过一个分数阈值进入下一关（共 5 关）
6. 撞墙或撞自己则游戏结束

---

## 关卡难度

| 关卡 | 需达到分数 | 蛇的基础速度 |
|------|-----------|------------|
| 1    | 50        | 120ms/步   |
| 2    | 120       | 110ms/步   |
| 3    | 220       | 100ms/步   |
| 4    | 350       | 90ms/步    |
| 5    | 500       | 80ms/步    |

每关开始时会重新随机生成 10 个障碍物和 3 个道具。
