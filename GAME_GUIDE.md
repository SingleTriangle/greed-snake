# Greed Snake - 游戏介绍

## 简介

Greed Snake 是一款复古像素风格的贪吃蛇游戏，采用 HTML5 Canvas 绘制，音效全部由 Web Audio API 程序化合成。游戏共 5 个关卡，难度递增，包含障碍物躲避和道具收集系统，适合休闲娱乐。

---

## 游戏规则

### 基本规则

- **目标**：操控蛇吃掉所有食物，累积分数，通过全部 5 个关卡
- **蛇的移动**：自动向前移动，玩家通过方向键或 WASD 控制方向
- **180° 掉头禁止**：蛇不能直接掉头（例如正在向右移动时，按左键无效）
- **蛇身增长**：每吃掉一个食物，蛇身增长 1 节

### 死亡条件

| 情况 | 结果 |
|------|------|
| 蛇头撞到墙壁 | 无影响，穿墙到对侧继续 |
| 蛇头撞到自己身体 | 蛇身缩短到碰撞点，缩短后蛇身为 0 时游戏结束 |
| 蛇身被障碍物撞到只剩 0 节 | 游戏结束 |

### 计分规则

- 每吃掉一个食物：**+10 分**
- 最高分记录在本地，最多保留一条

---

## 关卡设计

| 关卡 | 过关所需分数 | 蛇移动速度 |
|------|------------|-----------|
| 1    | 50 分      | 120ms/步  |
| 2    | 120 分     | 110ms/步  |
| 3    | 220 分     | 100ms/步  |
| 4    | 350 分     | 90ms/步   |
| 5    | 500 分     | 80ms/步   |

每进入新关卡，系统会重新随机生成 10 个障碍物和 3 个道具，同时蛇的移动速度略微提升。

---

## 道具系统

每关随机生成 3 个道具，收集后产生以下效果：

| 道具 | 颜色 | 效果 | 持续时间 |
|------|------|------|---------|
| 加速 | 金色圆形（标有 `+`） | 移动速度变为 1.5 倍 | 5 秒 |
| 减速 | 蓝色圆形（标有 `-`） | 移动速度变为 0.75 倍 | 5 秒 |

道具被收集后消失，同一关卡内不会补充。效果结束后速度自动恢复正常。

---

## 障碍物系统

- 每关固定生成 **10 个障碍物**
- 障碍物有两种类型：
  - **静态障碍物**：固定不动
  - **移动障碍物**：每 200ms 在网格内移动一格，碰到边界自动反弹
- 蛇头撞到障碍物：蛇身缩短 1 节；如果此时蛇身只剩 1 节，则游戏结束

## 操作说明

| 按键 | 功能 |
|------|------|
| `↑` 或 `W` | 向上移动 |
| `↓` 或 `S` | 向下移动 |
| `←` 或 `A` | 向左移动 |
| `→` 或 `D` | 向右移动 |
| `Space` | 开始游戏 / 重新开始 |
| `P` 或 `Esc` | 暂停 / 恢复 |

---

## 游戏界面元素说明

| 元素 | 颜色 | 描述 |
|------|------|------|
| 蛇头 | 荧光绿 | 带发光效果，是蛇的移动前端 |
| 蛇身 | 深绿渐变 | 颜色由头到尾逐渐变深 |
| 食物 | 粉红色 | 方块，会持续缩放呼吸动画 |
| 静态障碍物 | 灰色 | 固定不动，碰到会缩短蛇身 |
| 移动障碍物 | 灰色 | 会缓慢移动，需注意躲避 |
| 加速道具 | 金色圆形 `+` | 吃后速度提升至 1.5 倍 |
| 减速道具 | 蓝色圆形 `-` | 吃后速度降低至 0.75 倍 |

---

## 游戏状态

| 状态 | 说明 |
|------|------|
| START | 标题画面，按空格开始 |
| PLAYING | 正常游戏进行中 |
| PAUSED | 暂停，按 P 继续 |
| LEVEL UP | 过关动画，1.5 秒后自动继续 |
| GAME OVER | 死亡画面，显示最终得分 |
| WIN | 通关画面，恭喜完成全部 5 关 |
